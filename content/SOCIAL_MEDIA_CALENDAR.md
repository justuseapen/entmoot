## Crisis/Negative Feedback Protocol

### If Something Goes Wrong

1. **Acknowledge quickly**: Respond within 1 hour
2. **Take it offline**: "Let's discuss via DM/email"
3. **Investigate internally**: What actually happened?
4. **Resolve personally**: Direct founder/team involvement
5. **Follow up publicly**: "Issue resolved, thank you for the feedback"

### Negative Comment Response Template

```
"Thanks for sharing this feedback, [Name]. We take every user experience seriously. I'd love to understand more - could you DM us or email support@entmoot.app so we can make this right?"
```

### Never
- Argue publicly
- Delete comments (unless spam/abusive)
- Make promises you can't keep
- Blame the user
